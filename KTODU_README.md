## 1. სათაური

```
სამშენებლო მასალების ბიუჯეტი
```

> რაოდენობა (F) — `numFmt: "#,##0"` + `alignment: center`

## 5. ჯამის სტრიქონი

ბოლო სტრიქონი — ავტომატური SUM ფორმულა:

```
ws.getCell(`H${lastRow}`).value = { formula: `SUM(H4:H${lastRow - 1})` }
ws.getCell(`H${lastRow}`).numFmt = '#,##0.00'
```

Bold + მწვანე ფონი + border top medium.

## 6. დამატებითი პარამეტრები

- **Frozen header:** `worksheet.views = [{ state: 'frozen', ySplit: 3 }]`
- **ავტო-სიგანე:** `column.width = Math.max(header.length, 12)`
- **ყველა უჯრა:** `alignment: { vertical: 'middle' }`
- **Border:** `style: 'thin'` ყველა სტრიქონზე
- **თარიღი:** B2 უჯრა, ნახაზის სახელი A2
- **შრიფტი:** Calibri 11px / სათაური 16px Bold
